About srcAnalyzer

This project is powered by srcML, an open-source technology originally developed at Kent State University by Professor Jonathan I. Maletic and Michael L. Collard. srcML converts source code into structured XML, enabling deeper analysis of how code is written.

By leveraging srcML, srcAnalyzer provides fast and meaningful insights to help developers write cleaner, more maintainable, and more secure C++ code.

Developed at Kent Hack Enough 2026 by Roba Abdeta, Anthony Ross, Shagai Herdman, and Kalvik Ram Pemmasani.
