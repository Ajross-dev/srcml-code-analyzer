About srcAnalyzer

This project is powered by srcML, an open-source technology originally developed at Kent State University by Professor Jonathan I. Maletic and Michael L. Collard. srcML converts source code into a structured XML format, enabling powerful analysis, transformation, and a deep dive into code written by all developers.

By leveraging srcML, srcAnalyzer provides fast and meaningful insights, helping developers write cleaner, more maintainable, and secure code.

Developed here at the Kent Hack Enough 2026 Hackathon by Roba Abdeta, Anthony Ross, Shagai Herdman, and Kalvik Ram Pemmasani
